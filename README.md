# Miniguia de Estudos com NotebookLM

## Objetivo

O objetivo deste projeto foi utilizar o NotebookLM como ferramenta de apoio aos estudos sobre Large Language Models (LLMs), explorando recursos de Inteligência Artificial para organizar informações, gerar resumos e criar materiais de revisão.

---

## Fontes Utilizadas

1. https://en.wikipedia.org/wiki/Large_language_model
2. https://en.wikipedia.org/wiki/Transformer_(deep_learning_architecture)
3. https://en.wikipedia.org/wiki/Prompt_engineering

---

## Engenharia de Prompts

### Prompt 1

**Pergunta:**
> Explique o que é um Large Language Model como se eu nunca tivesse estudado Inteligência Artificial.

**Resultado:**
Recebi uma explicação introdutória sobre o funcionamento dos LLMs, sua capacidade de compreender linguagem natural e exemplos de aplicações.

---

### Prompt 2

**Pergunta:**
> Qual a diferença entre IA Generativa e um LLM?

**Resultado:**
A resposta mostrou que IA Generativa é um conceito mais amplo, enquanto os LLMs são um tipo específico de IA Generativa voltado para linguagem.

---

### Prompt 3

**Pergunta:**
> Faça um resumo em tópicos sobre todo o conteúdo das fontes.

**Resultado:**
Foi gerado um resumo organizado, facilitando a revisão do conteúdo.

---

## Dificuldades Encontradas

No início utilizei prompts muito genéricos, o que gerou respostas superficiais. Após tornar as perguntas mais específicas, obtive respostas mais completas e organizadas.

# Miniguia de Estudo

## O que são LLMs?

São modelos de Inteligência Artificial treinados com grandes volumes de texto para compreender e gerar linguagem natural.

## Como funcionam?

- Recebem um texto como entrada.
- Processam esse texto utilizando redes neurais.
- Identificam padrões aprendidos durante o treinamento.
- Geram uma resposta coerente.

## Principais aplicações

- Chatbots
- Tradução automática
- Geração de textos
- Programação
- Resumos
- Atendimento ao cliente

---

# Glossário

**LLM:** Large Language Model.

**Prompt:** Instrução enviada para a IA.

**Token:** Pequena unidade de texto utilizada pelo modelo.

**Transformer:** Arquitetura utilizada pelos LLMs modernos.

**IA Generativa:** Inteligência Artificial capaz de criar novos conteúdos.
